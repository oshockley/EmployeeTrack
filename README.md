# Description
EmployeeTrack is a command-line application built for users to manage a company's employee database, using Node.js, Inquirer, and PostgreSQL.

# Features
This command-line app allows users to manage company data by viewing, adding, and updating departments, roles, and employees. It presents formatted tables of departments, roles, and employee details, and lets users add new records or update an employeeâ€™s role in the database.
